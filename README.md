# Top10DownloaderApp
An Android App that downloads a RSS feed from Apple's website, displays the top 10/25 free apps, paid apps and songs in a list.

## Technologies used:
* Kotlin
* Android Studio
* XML
* Themes
* Material Design (Google)
* RxJava2
* MVVM architecture
* List Adapter
* Glide
* XmlPullParser
